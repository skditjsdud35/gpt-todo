# GPT Todo

This is a simple todo list application built with **React** and **TypeScript**. It demonstrates a minimal setup using Vite as the build tool.

## Features

- Add tasks to a list
- Press **Enter** to quickly add tasks
- Minimal styling for clarity

## Getting Started

To run this project locally, install dependencies and start the development server:

```bash
npm install
npm run dev
```

Then open your browser at [http://localhost:5173](http://localhost:5173) (the default Vite port) to view the app.

## Build for Production

To build the project for production deployment:

```bash
npm run build
```

The optimized build will be output to the `dist` directory.
